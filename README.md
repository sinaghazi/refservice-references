# Referenssipalvelu References — WordPress plugin

[![Latest release](https://img.shields.io/github/v/release/sinaghazi/refservice-references)](https://github.com/sinaghazi/refservice-references/releases/latest)

Display your company references from [Referenssipalvelu](https://referenssipalvelu.fi)
on your WordPress website. The plugin fetches references, products, and services from
your Referenssipalvelu account over its public API and renders them in grid, list, or
carousel layouts — as a Gutenberg block or via the `[refservice]` shortcode.

Requires WordPress 5.9+ and PHP 7.4+. See `readme.txt` for full feature and FAQ details.

## Installation

1. **[Download the latest release zip](https://github.com/sinaghazi/refservice-references/releases/latest/download/refservice-references.zip)**
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**, choose the
   downloaded `refservice-references.zip`, and click **Install Now**.
3. **Activate** the plugin.
4. Go to **Settings → Referenssipalvelu References** and enter the API key provided
   by your Referenssipalvelu account manager.
5. Add the **References** block to a page, or use the `[refservice]` shortcode.

## Updating

Updates are manual for now: download the latest zip from the link above, then upload
it via **Plugins → Add New → Upload Plugin**. WordPress detects the installed copy and
offers **Replace current with uploaded** — your settings are preserved.
